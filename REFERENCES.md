## نکات مهم

1. **همیشه از منابع رسمی استفاده کنید**: برای اطلاعات دقیق و به‌روز، همیشه به مستندات رسمی مراجعه کنید.

2. **نسخه‌ها را بررسی کنید**: مطمئن شوید که از مستندات نسخه صحیح استفاده می‌کنید.

3. **Stack Overflow**: برای مشکلات خاص و سوالات، Stack Overflow منبع خوبی است.

4. **GitHub**: برای مثال‌های کد و پروژه‌های open-source، GitHub را بررسی کنید.

5. **مستندات پروژه**: برای اطلاعات خاص این پروژه، به فایل‌های مستندات در پوشه پروژه مراجعه کنید.
